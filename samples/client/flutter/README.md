# A2UI Flutter client examples

To run examples from this folder, install Flutter SDK ([see docs](https://docs.flutter.dev/get-started/install)) and run:

```bash
cd <example_name>/app
flutter run -d web
```

## Supported platforms

Flutter packages are enabled and tested for two platforms: `macos` and `web`. To build and test on other platforms, follow [Flutter documentation](https://docs.flutter.dev/platform-integration).
